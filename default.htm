    Professor

    My research interests and projects are in programming languages, compilers, and programming environments.

    C Interfaces and Implementations: Techniques for Creating Reusable Software (Addison-Wesley, August 1996).

    Some of my recent work centers on lcc, a retargetable, production compiler for ANSI C. lcc is joint work with Chris Fraser, and it's described in our book A Retargetable C Compiler: Design and Implementation.

    Software

    iburg
    A program that generates fast tree parsers for cost-augmented tree grammars. Joint work with Chris Fraser and Todd Proebsting.
    duel
    A very high-level debugging language. duel is implemented as an extension to gdb. Joint work with Michael Golan.

    More software is available from http://www.cs.princeton.edu/software/ or by anonymous ftp from ftp.cs.princeton.edu.
