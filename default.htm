    • David R. Hanson
    Professor

    My research interests and projects are in programming languages, compilers, and programming environments. Some of my recent work centers on lcc, a retargetable, production compiler for ANSI C. lcc is joint work with Chris Fraser.

    You'll undoubtedly want to buy copies of my books:
    A Retargetable C Compiler: Design and Implementation, with Chris Fraser (Addison-Wesley, 1995, ISBN 0-8053-1670-1)
    C Interfaces and Implementations: Techniques for Creating Reusable Software (Addison-Wesley, 1997, ISBN 0-201-49841-3)

    Software

    iburg A program that generates fast tree parsers for cost-augmented tree grammars. Joint work with Chris Fraser and Todd Proebsting.
    duel A very high-level debugging language. duel is implemented as an extension to gdb. Joint work with Michael Golan.

    More software is available from http://www.cs.princeton.edu/software/ or by anonymous ftp from ftp.cs.princeton.edu.
