David R. Hanson

My research interests and projects are in programming languages, compilers, and programming environments. Some of my recent work centers on lcc, a retargetable, production compiler for ANSI C. lcc is joint work with Chris Fraser. lcc is a component of the Zephyr project, which is part of the National Compiler Infrastructure.

"Early Experience with ASDL in lcc", MSR TR-98-50, 9/98,  (126KB). Talk slides 6/19/98. (71KB).

Software

cdb A machine-independent debugger for C programs compiled by lcc 4.x. With Mukund Raghavachari.
iburg A program that generates fast tree parsers for cost-augmented tree grammars. With Chris Fraser and Todd Proebsting.
duel A very high-level debugging language. duel is implemented as an extension to gdb. With Michael Golan.
